Atividade CSS

Como surgiu o CSS?

O CSS (Cascading Style Sheets), ou Folhas de Estilo em Cascata, surgiu da necessidade de separar a estrutura e o conteúdo de documentos HTML de sua apresentação visual. No início da web, por volta de 1990, os desenvolvedores usavam o próprio HTML para definir o estilo das páginas, utilizando elementos como e atributos inline, o que dificultava a manutenção e a reutilização de código. A proposta do CSS foi apresentada em 1994 por Håkon Wium Lie, que trabalhava na CERN (a mesma organização onde nasceu a World Wide Web). Na época, ele buscava uma maneira eficiente de aplicar estilos de forma consistente e centralizada em vários documentos HTML. Juntamente com Bert Bos, eles desenvolveram a especificação inicial do CSS. Em 1996, o World Wide Web Consortium (W3C) publicou a primeira recomendação oficial do CSS, o CSS1. Essa versão permitia definir cores, fontes, espaçamentos e alinhamentos de texto, além de introduzir o conceito de "cascata", onde múltiplas regras de estilo são aplicadas com base em uma hierarquia de prioridades. Com o tempo, o CSS evoluiu com novas versões (CSS2, CSS2.1, CSS3 e além), incorporando recursos mais avançados, como layouts flexíveis (Flexbox), grids, animações e variáveis. Essa evolução consolidou o CSS como uma ferramenta essencial no desenvolvimento web moderno, promovendo uma separação clara entre conteúdo (HTML) e estilo (CSS), o que melhora a manutenibilidade, a acessibilidade e o desempenho das páginas web.

Onde foi feito o primeiro uso do CSS?

O primeiro uso prático do CSS ocorreu em meados da década de 1990, logo após sua proposta inicial por Håkon Wium Lie, em 1994. O primeiro navegador a oferecer suporte parcial ao CSS foi o Internet Explorer 3, lançado pela Microsoft em 1996. Esse suporte coincidia com a publicação da primeira especificação oficial do CSS (CSS1) pelo W3C no mesmo ano. O Internet Explorer 3 permitiu que os desenvolvedores começassem a aplicar estilos separados do HTML, embora de forma limitada e com muitas inconsistências em relação ao padrão. Mesmo assim, foi um marco importante, pois abriu caminho para a adoção gradual do CSS no desenvolvimento web. Pouco tempo depois, outros navegadores como Netscape Navigator começaram a incorporar suporte ao CSS, mas também com muitas diferenças na forma como interpretavam as regras. Essa falta de uniformidade tornou o início da adoção do CSS um desafio, exigindo soluções alternativas e adaptações específicas para cada navegador. Apesar das dificuldades iniciais, o CSS foi ganhando força à medida que os navegadores evoluíam e os benefícios da separação entre conteúdo e apresentação ficavam mais claros. Com o tempo, o CSS se tornou padrão essencial para o design de páginas web modernas.
